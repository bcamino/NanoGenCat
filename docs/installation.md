# Installation

## Environment and dependences

* `bash Miniconda3*.sh`
* `conda create -n bcnm`
* `conda activate bcnm`
* `pip install --upgrade pip` - Upgrade pip
* `pip install ase pyyaml pandas pymatgen` - Install dependences 

## Download BcnM

* `cd NanoGencat`
* `conda activate bcnm`
* `python3 bcnm.py examples/ruthenium.input.yaml`
